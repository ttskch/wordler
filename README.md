# Wordler

A [Wordle](https://www.powerlanguage.co.uk/wordle/) solver implemented with [symfony/panther](https://github.com/symfony/panther).

## Installation

```shell
$ git clone git@github.com:ttskch/wordler.git
$ cd wordler
$ composer install
```

## Usage

```shell
$ bin/wordler
```
